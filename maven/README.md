## 🚀 Quick Commands
```bash
mvn archetype:generate    # Create new project
mvn clean package         # Clean + build JAR/WAR
mvn test                  # Run all tests
mvn help:effective-pom    # See the final merged POM
mvn install               # Copy to local ~/.m2 repo
```

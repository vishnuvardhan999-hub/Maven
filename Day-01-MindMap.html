📚 Key Concepts

DEFINITION

🔧 What is Maven?

Free and open-source build automation tool developed by the Apache Software Foundation. Manages the entire build lifecycle from compilation to deployment.

  • Creates standard project folder structure
  • Downloads and manages dependencies
  • Runs unit tests automatically
  • Packages into JAR or WAR files

REPOSITORIES

🏪 3 Types of Repositories

  • Local — ~/.m2/repository (your machine cache)
  • Central — repo1.maven.org (official online)
  • Remote — Company/private server

First time → downloads from Central. Next time → uses Local directly ⚡

COMMANDS

⌨️ Key Maven Commands

mvn archetype:generate # Create project mvn compile # Compile code mvn test # Compile + test mvn package # Compile+test+JAR mvn clean package # Clean then package mvn install # Copy to ~/.m2 mvn deploy # Push to remote repo

ARCHETYPES

🏗️ Archetype Templates

A template that defines the structure of a Maven project.

ArchetypeCreates
maven-archetype-quickstartJava standalone app
maven-archetype-webappJava web app (WAR)
mvn archetype:generate \ -DarchetypeArtifactId=\ maven-archetype-quickstart
GOALS

🎯 Maven Goals Explained

  • clean — deletes target/ folder
  • compile — .java → .class
  • test — runs JUnit tests
  • package — creates JAR/WAR
  • install — copies to ~/.m2
  • deploy — uploads to remote
